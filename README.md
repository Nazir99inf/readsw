# WhatsApp Auto-Reactor & AI Chat Bot

Skrip bot WhatsApp berbasis **Baileys** yang dilengkapi dengan dua fitur utama:

- **Auto React** pada WhatsApp Story.
- **Chat AI** pada pesan pribadi.
